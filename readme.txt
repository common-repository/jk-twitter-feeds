== Description ==

This is a Wordpress plugin that fetches twitter feeds from your page and show it on your website. Also the plugin features a widget with which the twitter feeds can be shown on the sidebar.

== Installation ==

This will guide you through the installation of the plaugin.

1.Activate the plugin through the 'Plugins' menu in WordPress.
2.Update the details in the "Twitter Feeds" tab in the adminpanel.
3.Use this function show_jk_twitter_feeds() to display twitter feeds.
4.Or Add the Widget "JK Twitter Widget" in your Wordpress sidebar.
